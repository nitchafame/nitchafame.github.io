Melody

Interactive 3D piece

Role : Designer, 3D modeling, C# Code

2014

 

Credits

Special Thanks - Robert Yang

 

"Melody" project is a sound interaction using the pentatonic scale (a musical scale with five notes) to make people enjoy creating sounds or a song with visuals. The user can press keys A, S, D, F, J, K to choose notes and press arrows up and down to change speed. Every time the user presses a key a stair is built.

 

Each block contains a melody, which has different melodies in different colors. The visual will build depending on how many keys that user chooses. Therefore allowing the user to build a new song, visual, and overall world every time they play.

 

The project stems from the concepts of “making and experiencing” discussed in "being in the world" documentary. It portrays the idea that people should experience and take risks in their world and the arts. Don’t be afraid to create things.
